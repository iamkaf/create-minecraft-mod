# create-minecraft-mod

A command-line tool to generate a new Minecraft mod project from a template.

## Usage

Run the tool and follow the prompts:

```
npx create-minecraft-mod
```

Youâ€™ll be asked to choose things like:

* Mod name and ID
* Package name
* Minecraft version
* Mod loaders
* Optional libraries
* Optional utility mods
* License

A project folder will be created and prepared based on your selections.

## Requirements

* Node.js 24 or later
